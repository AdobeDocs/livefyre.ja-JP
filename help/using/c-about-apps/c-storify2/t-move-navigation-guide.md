# ナビゲーションガイドの移動{#move-the-navigation-guide}

画面上のナビゲーションガイドの場所を指定します。

1. Storify 2 App Designerで、切り替えをオンにし **[!UICONTROL Navigation Guide]** ます。
1. ドロップダウンから上、左、右のいずれかを **[!UICONTROL Orientation]** 選択します。
