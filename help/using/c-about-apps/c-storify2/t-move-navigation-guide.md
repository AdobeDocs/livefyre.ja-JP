# ナビゲーションガイドを移動{#move-the-navigation-guide}

画面上のナビゲーションガイドの場所を指定します。

1. Storify 2 App Designerで、**[!UICONTROL Navigation Guide]**&#x200B;の切り替えをオンにします。
1. **[!UICONTROL Orientation]**&#x200B;ドロップダウンから上、左、右のいずれかを選択します。
