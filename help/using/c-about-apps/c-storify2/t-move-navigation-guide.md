# ナビゲーションガイドの移動{#move-the-navigation-guide}

画面上のナビゲーションガイドの場所を指定します。

1. Storify2App Designerで **[!UICONTROL Navigation Guide]** 、切り替えを切り替えます。
1. **[!UICONTROL Orientation]** ドロップダウンから上、左または右を選択します。
