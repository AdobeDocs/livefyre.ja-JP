# Storify 2追加{#add-a-navigation-guide-to-a-storify}のナビゲーションガイド

1. Storify 2ストーリーを開きます。
1. Storify 2 App Designerで、**[!UICONTROL Navigation Guide]**&#x200B;の切り替えをオンにします。
