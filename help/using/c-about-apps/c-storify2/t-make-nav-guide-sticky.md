# ナビゲーションガイドを固定する{#make-the-navigation-guide-sticky}

ナビゲーションガイドを画面の特定の領域に合わせることを選択します。

1. Storify2App Designerで **[!UICONTROL Navigation Guide]** 、切り替えを切り替えます。
1. **[!UICONTROL Stickiness]** 切り替えを切り替えます。
