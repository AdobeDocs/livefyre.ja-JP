# 投票{#polls}

顧客がリアルタイムで意見を述べることを許可します。

投票を使用すると、質問が1回だけの投票を作成して、サイトにすばやく投稿できます。 調査を使用して、読者の参加を促し、フィードバック、意見を収集し、サイト訪問者からのリクエストを行います。

>[!NOTE]
>
>投票はStreamにリンクされません。
