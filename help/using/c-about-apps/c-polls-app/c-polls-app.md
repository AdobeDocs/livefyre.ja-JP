# 投票{#polls}

クライアントが自分の意見をリアルタイムで伝える。

調査では、質問が1つだけの調査を作成してサイトに投稿できます。 調査を使用して、読者の関与を促し、フィードバック、意見を収集し、サイトの訪問者からのリクエストを行います。

>[!NOTE]
>
>調査がストリームにリンクされません。
