# トレンド機能{#trending-features}

「傾向分析」オファーでは、これらの機能がサイトで使用されます。



* 「トレンド」は、どのコレクションが最もアクティブかに関する情報をリアルタイムで表示します。
* 会話の人気が最も高いものから最も低いものへと順に表示されます。
