# トレンド機能{#trending-features}

トレンドアプリでは、サイトに対してこれらの機能を提供します。



* 「トレンド」は、どのコレクションが最もアクティブかに関するリアルタイム情報を表示します。
* 会話は最も人気の高いものから最も人気の低いものへと順に表示されます。
