# トレンドの機能{#trending-features}

トレンドアプリでは、サイトのこれらの機能が提供されます。



* トレンドには、コレクションが最もアクティブなリアルタイム情報が表示されます。
* 会話は最も人気の高いものから最も少ないものへと一覧表示されます。
