# Map Features{#map-features}

マップアプリは、サイトのこれらの機能を提供します。



* コンテンツはマップにリアルタイムで入力されます。
* 同じ領域のコンテンツがクラスター化され、ギャラリーモーダルで表示できます。
* マップにズームインすると、特定の領域のコンテンツに焦点を合わせます。
