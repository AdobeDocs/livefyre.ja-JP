# マップ機能{#map-features}

Mapアプリでは、サイトに次の機能を提供します。



* コンテンツがリアルタイムでマップに入ります。
* 同じ領域のコンテンツは集積され、ギャラリーモーダルで表示できます。
* Mapにズームインして、特定の領域のコンテンツに焦点を当てます。
