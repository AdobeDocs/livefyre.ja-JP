livefyre. jsを使用して認証を設定します。詳しくは、Livefyre. jsを使用したアプリへの認証 [の追加を参照してください](/help/implementation/c-getting-started/c-implementation-process/c-using-livefyre.js-to-create-customize-and-use-apps-on-your-site.md)
