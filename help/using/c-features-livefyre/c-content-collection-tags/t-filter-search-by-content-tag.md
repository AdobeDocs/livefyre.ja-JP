# コンテンツタグで検索をフィルター{#filter-search-by-content-tag}

Studio内のコンテンツ検索をコレクションタグでフィルタリングします。

1. **[!UICONTROL Content]**&#x200B;パネルから&#x200B;**[!UICONTROL Filter by: Contains]**&#x200B;を選択します。
1. タグを入力します。
