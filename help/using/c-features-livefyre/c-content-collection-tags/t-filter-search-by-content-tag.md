# コンテンツタグで検索をフィルター{#filter-search-by-content-tag}

Studioでのコンテンツ検索をコレクションタグでフィルタリングします。

1. パネル **[!UICONTROL Filter by: Contains]** からを選択 **[!UICONTROL Content]** します。
1. タグを入力します。
