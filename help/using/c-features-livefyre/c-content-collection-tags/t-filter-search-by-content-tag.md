# コンテンツタグで検索をフィルター{#filter-search-by-content-tag}

Studioでコンテンツ検索をコレクションタグ別にフィルターします。

1. パネル **[!UICONTROL Filter by: Contains]** から選択 **[!UICONTROL Content]** します。
1. タグを入力します。
