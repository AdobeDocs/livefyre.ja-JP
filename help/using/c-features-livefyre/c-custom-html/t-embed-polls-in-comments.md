# コメントに投票を埋め込む{#embed-polls-in-comments}

コメントストリームまたはライブブログストリームに調査を埋め込みます。

1. StudioのApp Managerページで、（埋め込み）ボタンをク **[!UICONTROL </>]** リックして、ポーリングの埋め込みコードを生成します。
1. 埋め込みコードをコピーして、アプリのフィー **[!UICONTROL Add Custom HTML]** ルドに貼り付けます。
