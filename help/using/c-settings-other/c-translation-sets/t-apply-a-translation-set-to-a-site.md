# 翻訳セットのサイトへの適用{#apply-a-translation-set-to-a-site}

**翻訳セットのサイトへの適用**

1. Studioの設定に移動します。
1. サイトを選択します。
1. 「サイトの設定」を選択します。
1. 「アプリ設定」を選択します。
1. デフォルト設定/翻訳セットに移動します。
1. ドロップダウンメニューから翻訳セットを選択します。
