# プライバシーレポートの表示{#view-a-privacy-report}

Livefyreでプライバシーレポートを表示して、ユーザーのLivefyreに記録された情報を確認します。

1. 移動 **[!UICONTROL Settings > Privacy]**.
1. リストからプライバシーレポートをクリックして表示します。
