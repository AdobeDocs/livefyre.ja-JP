# 表示aプライバシーレポート{#view-a-privacy-report}

Livefyreでプライバシーレポートを表示して、ユーザーのLivefyreに記録されている情報を確認します。

1. 移動 **[!UICONTROL Settings > Privacy]**.
1. リストから表示にプライバシーレポートをクリックします。
