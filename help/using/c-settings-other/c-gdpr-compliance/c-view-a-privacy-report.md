# プライバシーレポートの表示{#view-a-privacy-report}

Livefyreでプライバシーレポートを表示して、ユーザーのLivefyreにどの情報が記録されているかを確認します。

1. 移動 **[!UICONTROL Settings > Privacy]**先
1. リストからプライバシーレポートをクリックして表示します。
