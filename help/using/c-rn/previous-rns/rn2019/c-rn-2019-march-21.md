# 2019年3月22日更新のリリースノート

>[!NOTE]
>

## 機能強化

* デフォルトで、「行動喚起」ボタンの有効化が実装されました。

* ModQ内の「未定義」のInstagram投稿を削除できない問題を修正しました。
