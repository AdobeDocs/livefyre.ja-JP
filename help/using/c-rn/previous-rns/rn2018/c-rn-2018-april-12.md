# 2018年4月12日{#april}

2018年4月12日リリースのリリースノートです。

## 新機能 {#section_syx_mdt_wcb}

このリリースの実稼働版では、次の機能が新しく追加されました。

ストリーム内のスマートタグの信頼性のしきい値を選択できるようになりました。 タグの精度スコア(0 ～ 100)を設定すると、取得するアセットの精度を制御できます。
