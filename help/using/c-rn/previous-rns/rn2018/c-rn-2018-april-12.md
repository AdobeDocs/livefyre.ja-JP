# 2018年4月13日{#april}

2018年4月13日リリースのリリースノートです。

## 新機能 {#section_syx_mdt_wcb}

このリリースの実稼働版では、次の機能が新機能です。

ストリーム内のスマートタグの信頼性しきい値を選択できるようになりました。タグに精度スコア（0~100）を設定すると、取得するアセットの正確性を制御できます。
