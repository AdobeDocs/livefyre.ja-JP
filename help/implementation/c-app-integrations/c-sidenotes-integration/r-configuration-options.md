# サイト設定オプション{#sidenotes-configuration-options}

simees configオブジェクトは、Livefyreアプリケーションがページ上でレンダリングするコンテンツを指定するために使用するJSONオブジェクトです。

このオブジェクトには、次のパラメーターが含まれています。

| パラメータ | タイプ | 説明 |
|--- |--- |--- |
| AuthDelegate | *必須* オブジェクト | 新しいスタイル化されたまたは古いスタイルの認証デリゲート。 |
| CollectionMeta | *必須* オブジェクト | 詳しくは、CollectionMetaトークンを参照してください。 |
| customIcon | *オプション* の文字列 | カスタムランチャーアイコンのURLを設定します。初期設定はLivefyre吹き出しです。 |
| customStyles | *オプション* オブジェクト | Sideingのルックアンドフィールを変更するカスタムスタイルを追加します。詳しくは、「カスタムスタイル」を参照してください。 |
| disableStream | *オプション* のブール値 | trueの場合、このSideesインスタンス内のリアルタイムストリーミングを無効にします（他のコメントストリームでは無効）。初期設定はfalseです。 |
| 環境 | *オプション* の文字列 | Livefyre UAT環境を指定します。可能な値 `t402.livefyre.com`は、のみです。実稼働環境では、このパラメーターを削除する必要があります。 |
| iconVisibility | *オプション* のオブジェクトまたは文字列 | カーソルを合わせたとき、または静的に表示するかどうかを定義します。これが文字列の場合、ブロックアイコンの両方のバージョンに使用されます（空のものとSareを持つ）。オブジェクトの場合、各タイプを指定する必要があります。{empty:"hover"， num:"static"}。初期設定はstaticです。 |
| inlineThreads | *オプション* のブール値 | trueの場合、Socialのスレッドは関連付けられたブロックの直後に挿入されます。初期設定はfalseです。 |
| numSiteNotSel | *オプション* のオブジェクト、文字列または配列 | カウントするカウントウィジェットの要素を指定します。（このウィジェットには、ページ上での閲覧数とsimeingに関する情報が表示されます）。文字列セレクターが複数の要素に一致する場合、最初の要素が選択されます。（CSS3DOMセレクター仕様を使用） |
| position | *オプション* の文字列 | simeingが有効になっている要素がクリックされたときのポップアップの位置を判定します。指定できる値は"smart"、"left"、"right"、"bottom"です。初期設定は"smart"です。スペースが足りない場合（コンテンツの下に移動する場合）は、ページの右側にポップアップが揃えられます。 |
| セレクター | *必須* オブジェクト、文字列または配列 | ランチャーアイコンを表示する要素を指定します。（CSS3DOMセレクター仕様を使用）オブジェクトの場合、"include"セクションと"exclude"セクションを含めます。これらのセクションには、一致するすべての要素にCSSセレクターが適用され、除外に一致する要素が削除されます。文字列の場合、ページに一致するエレメントが含まれます。配列の場合、含める要素をリストします。 |
| 文字列 | *オプション* オブジェクト | アプリケーション全体の言語またはテキストを変更するカスタムテキスト文字列を追加します。詳しくは、カスタム文字列を参照してください。 |
| ThreadContainerEL | *オプション* のオブジェクト、文字列または配列 | 一重引用符のスレッドが表示される要素を指定します。このパラメーターを使用すると、スレッドを再配置できます。文字列セレクターが複数の要素に一致する場合、最初の要素が選択されます。（CSS3DOMセレクター仕様を使用） |
