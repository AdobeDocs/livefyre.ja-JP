# サイドブックマークレット{#sidenotes-bookmarklet}

サイドは、既存のページに読み込むことができます（ほとんど）。

サイドのデモを行うには、次のブックマークレットを使用します。

サイドブックマークレット

ブックマークレットをブックマークにコピーし、リンクをクリックしてサイドを読み込みます。 現在のページ上のサイドの装飾に使用するセレクターの入力を求められます。
