# BuildLiveFyReTokenネットワークメソッド{#buildlivefyretoken-network-method}

呼び出されたネットワーク用の他のLivefyre APIとの対話に使用できる暗号化有効なLivefyreトークンを返します。

呼び出されたネットワーク用の他のLivefyre APIとの対話に使用できる暗号化有効なLivefyreトークンを返します。

デフォルトでは、このトークンは作成時点から24時間で有効期限が切れます。

```
network.buildLivefyreToken(); 
```

