# getUrnサイトメソッド{#geturn-site-method}

このメソッドは、このサイトのURNを返します。

## Javaの例{#section_nyl_ycs_rz}

用途:

```
site.getUrn();
```

出力例：

```
"urn:livefyre:network=`example.fyre.co`:site=1" 
```

## NodeJSの例{#section_xkd_gds_rz}

