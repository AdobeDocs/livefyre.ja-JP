# アプリから機能するコンテンツを選択{#select-content-to-feature-from-an-app}

アプリから直接コンテンツの機能を有効または無効にします。

1. アプリ内で、機能させるコンテンツの上にカーソルを置きます。
1. をクリッ **[!UICONTROL Feature]** クして、コンテンツを特集 **[!UICONTROL Unfeature]** するか、そのコンテンツの特集を中止します。
